# Rock Paper Scissors
**🔗 Live demo:** [rock-paper-scissors-kappa-topaz.vercel.app](https://rock-paper-scissors-kappa-topaz.vercel.app/)
A browser-based Rock Paper Scissors game built with vanilla HTML, CSS, and JavaScript — play against the computer using keyboard shortcuts, track your win/loss/tie record, or let it autoplay.

## Features
- Keyboard-controlled gameplay: press `r` for rock, `p` for paper, `s` for scissor
- Live win / loss / tie stats tracker
- Autoplay mode that simulates continuous random rounds
- Reset button to clear stats

## Tech Stack
HTML · CSS · Vanilla JavaScript (no frameworks)

## Getting Started
Clone the repo and open `printQuantity.html` directly in your browser:
```bash
git clone https://github.com/uryapraveen/Rock-Paper-Scissors.git
cd Rock-Paper-Scissors
```
Then open `printQuantity.html` in any browser.
